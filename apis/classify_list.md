# 分类管理
## 国家分类
#### 请求地址
    /api/classify/city
#### 请求方式
    GET
#### 请求参数
    {
        city_type: "中国品牌", //国家品牌:中国品牌 [字符串:必填]
        brand_list: "brand" //品牌列表:欧碧泉 [字符串:必填]
    }
#### 返回数据
    {
        "errcode":0, // 错误代码: [数值:必填] 0 无错误 -1 有错误
        "errmsg":"", //错误信息: [字符串:默认为空]
        "data": { //数据内容
            start: 0,
        }
    }
